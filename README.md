# Verion 🚀

Verion is a modern browser-based cloud coding platform that allows users to create, edit, preview, and deploy web projects directly from the browser without any setup.

## ✨ Features
- **Online Code Editor:** Powered by Monaco (VS Code engine) for a native feel.
- **Real-Time Preview:** Instant visual feedback for fast iteration.
- **Autonomous AI Assistant:** Built-in Google Gemini integration for smart code generation, multi-file diffs, and debugging.
- **One-Click Deployment:** Instantly push code to live URLs via Vercel API.
- **Cloud Storage:** Secure user authentication and project persistence.
- **Minimalist UI:** Fast, intuitive, and highly focused environment.

## 🛠 Tech Stack
- **Frontend:** Next.js (React), TailwindCSS, Monaco Editor
- **Backend:** Next.js API Routes, Vercel Deployment API
- **Database & Auth:** Supabase (PostgreSQL)
- **AI Engine:** Google Gemini 1.5 API
- **Payments:** Razorpay (Upcoming)

## 🎯 Purpose
Verion is designed for students, beginners, and developers who want a lightweight, fast, and powerful cloud coding environment without the hassle of installing software.
